# 🏆 SportsApp - Automated Sports Photography

Este es un proyecto independiente enfocado en la automatización del procesamiento de fotografías deportivas.

## ✨ Características
*   **Identificación de Jugadores:** Uso de OCR para detectar números en shorts y camisetas.
*   **Recorte Automático:** Centrado de rostro inteligente para retratos de atletas.
*   **Overlays Dinámicos:** Aplicación de gráficos de marca y banners de información.
*   **Optimización para Redes Sociales:** Generación de contenido listo para publicar.

## 📂 Estructura
*   `index.html`: Interfaz de usuario para carga y previsualización.
*   `style.css`: Estilos modernos y deportivos.
*   `app.js`: Lógica de procesamiento de imágenes y detección.
